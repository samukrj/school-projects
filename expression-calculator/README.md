# Expression Calculator (C)

A simple school project written in C.  
The program evaluates basic arithmetic expressions containing digits, operators (+ − * /), and parentheses.  
It uses a small recursive function to process nested expressions.

## Features
- Input validation  
- Support for parentheses  
- Recursive evaluation  
- Basic operators: + − * /

## Example

Input: `8+(4*7)`  
Output: `36`

## How to run

**Compile:**
gcc myCalc.c -o myCalc

**Run:**
./myCalc
